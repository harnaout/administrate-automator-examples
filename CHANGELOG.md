## [1.0.0] - 2026-02-02

### Added
- Initial repository setup
- **Scheduler - Bulk Event Activation** workflow (v1.0.0)
  - Web form interface for bulk activating draft events
  - GraphQL integration for event management
  - Error handling and validation
- **Instructor Jump Ball** workflow (v1.0.0)
  - RSVP invitation system for instructors
  - Accept/Decline response handling
  - Automatic instructor assignment on acceptance
  - Internal notes logging with timestamps
- Comprehensive README for each workflow
- Repository structure and contribution guidelines
- Main README with workflow catalog

## Version Format

Workflow versions follow semantic versioning:
- **Major (X.0.0)**: Breaking changes or complete workflow redesign
- **Minor (0.X.0)**: New features, backward compatible
- **Patch (0.0.X)**: Bug fixes, minor improvements
